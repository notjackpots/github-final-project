# 🤝 How to Contribute

[![Contributions welcome](https://img.shields.io/badge/contribute-join-brightgreen?style=for-the-badge)](https://github.com/notjackpots/github-final-project/issues)

## Quick Start

1. **Fork** → **Clone** → **Branch** (`git checkout -b fix/bug`)
2. **Code** → **Test** → **Commit** (clear messages)
3. **PR** with checklist below

## What We Need

| Type | Examples |
|------|----------|
| 🐛 | Bugs, edge cases |
| ✨ | Features, UX |
| 📝 | Docs, typos |
| 💡 | Ideas, suggestions |

## Before PR

- [ ] Check open issues/PRs
- [ ] Small, focused changes  
- [ ] Tests pass
- [ ] Docs updated
- [ ] Screenshots if UI

## Questions?
Browse `good first issue` labels or ask here 👇

**Thanks for helping! 🚀**
