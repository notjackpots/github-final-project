# Simple Interest Calculator

A small calculator that computes **simple interest** using the principal amount, annual interest rate, and time in years.

## Formula

```text
Simple Interest = (P × R × T) / 100
```

## Inputs

- `P` — Principal amount
- `R` — Annual rate of interest
- `T` — Time period in years

## Output

- Simple interest value

## Example

If `P = 1000`, `R = 5`, and `T = 2`, then:

```text
Simple Interest = (1000 × 5 × 2) / 100 = 100
```
